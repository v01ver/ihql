# IHQL Is Helpful Query Language

IHQL is inspired by SQL and JSON-Schema.

Ref:

  * http://www.mathcs.emory.edu/~cheung/Courses/554/Syllabus/5-query-opt/SQL-grammar.html

## Syntax

        ihql:{a,t,s,f,w,j,r,o,v}

        a:crud
        o:object
        t:type of object
        l:list of object
        s:{o,l,d,c)
        f:n
        w:c,o,a
        j,p:boolean,k:boolean,r:[]
        r:g,o
